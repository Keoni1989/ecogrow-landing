# EcoGrow – Product Landing Page

A responsive landing page designed to present the concept of EcoGrow, a smart indoor gardening system focused on simplicity, automation and modern design.

## Why I built it

I wanted to practice product presentation by designing a landing page that combined visual hierarchy, storytelling and responsive design around a fictional product.

## Tech Stack

- HTML
- CSS
- JavaScript
- Vercel

## Status

Completed concept project.
